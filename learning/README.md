# Exercícios de aprendizagem

Exercícios curtos para consolidar os fundamentos do roadmap, sem virarem projetos paralelos de produção.

- `python/`: fundamentos, POO e pequenos programas de linha de comando.
- `pytest/`: percurso guiado de testes; consulte `pytest/AGENTS.md` e `pytest/PROGRESS.md`.
- `fastapi/`: experimentos pequenos de rotas e parâmetros.

O código destes diretórios é feito manualmente como parte do estudo. Um projeto original será definido apenas depois, quando houver base suficiente para sustentá-lo.
